# HARP: Haptic-Acoustic Resonance Platform

## 💡 一句话总结 (Elevator Pitch)

本项目旨在研发一款**基于柔性FPC孤岛与频分复用技术的超声-触觉双向交互手套**。该系统利用单个硅麦克风同时实现“超声共振触觉增强的闭环控制”与“3D手部姿态追踪”，并构建了一个具有高度理论研究价值的二阶微机电共振模型。

---

## 🔍 核心科研思路与创新点

### 1. 硬件架构与理论模型：优雅的“弹簧-质量-阻尼”二阶机械系统

* **硬件设计**：手套采用柔性印制电路板（FPC）制成，通过“FPC蛇行走线”将一个个“FPC孤岛”互相连接，每个孤岛上搭载一颗微型硅麦克风。
* **理论深度**：这一物理结构天然构成了一个**二阶机械振动系统**。其中，FPC孤岛与硅麦克风构成了**质量块 (Mass)**，连接孤岛的蛇形走线等效为**机械弹簧 (Spring)**，而空气阻力与手掌皮肤的粘弹性则构成了**阻尼器 (Damper)**。这为整个系统提供了极具深度的力学推导、共振模态分析和理论建模空间。

### 2. 触觉渲染机制：超声驱动的局部机械共振增强

* 外部的超声相控阵将 40kHz 的聚焦超声波精准打在指定的硅麦克风及对应的 FPC 孤岛上。
* 利用声辐射力激发孤岛产生结构共振。由于系统处于共振状态，微小的超声能量被机械系统大幅放大，从而激活手掌心对应位置的皮肤感受器，产生**强烈且高度局域化的触觉反馈**。

### 3. 核心技术亮点：基于单传感器的频分复用 (FDM)

该 Idea 最精妙之处在于**“一传感两用”**。由于硅麦克风具备极宽的拾音频带，系统在运行时，麦克风会同时捕获两种信号：

* **高频信号**：外部相控阵发出的 40kHz 超声波信号。
* **低频信号**：FPC 孤岛发生机械共振时产生的 200-300Hz 振动音频。
* **创新点**：无需增加额外的传感器，仅通过简单的两组带通滤波器（Band-pass Filter），即可将这两种频率截然不同的信号完美分离，实现**硬件极简、功能翻倍**的频分复用采集。

### 4. 双线并行的系统功能应用

分离出的两种信号，分别被赋予了极具价值的系统级功能：

* **低频应用（闭环触觉控制）**：利用提取出的 200-300Hz 共振信号，系统可以实时监测当前的振动幅度。通过**闭环反馈控制**，动态微调超声相控阵的输出，使 FPC 孤岛始终维持在最大共振频率点，从而保证触觉反馈强度的最大化与稳定性。
* **高频应用（3D空间位姿追踪）**：利用提取出的 40kHz 超声波信号，计算各个孤岛接收到信号的强度（Amplitude）或时间延迟（Time-of-Flight, ToF）。结合手部的运动学关节模型，能够精准解算出整个手部以及每一个手指关节在三维空间中的**绝对位置与姿态 (Spatial Localization & Posture Recognition)**。

---

## 🎯 总结与科研价值 (Impact)

这个 Idea 的绝佳之处在于它的**高集成度与多物理场耦合**。它打破了传统 VR/AR 手套“触觉致动器”与“空间追踪传感器”必须分离的硬件冗余痛点。
依靠优雅的二阶力学设计和巧妙的频分复用信号处理，仅用**“一个硅麦克风+一片FPC”**就形成了一个**“感知（姿态追踪）+ 行动（触觉反馈增强）+ 优化（闭环共振控制）”**的完整闭环。该研究不仅在人机交互（HCI）应用层面具有极高的商业潜力，在微机电系统（MEMS）的力声耦合理论分析上也具有充足的学术发掘空间。
